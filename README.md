📚 Room: Common Attacks

Status: Completed ✅
Platform: TryHackMe

🧠 Overview

This room introduced common cyber attack methods used by threat actors and how to recognise and defend against them. It focused on both technical and human-based attacks.

🎯 Key Topics & Breakdown
🔹 Social Engineering

What it is:
Manipulating people into giving away sensitive information.

Examples:

Pretexting
Baiting
Impersonation

Detection:

Unusual requests for sensitive data
Pressure or urgency tactics

Mitigation:

Security awareness training
Verification of identity before sharing information
🔹 Phishing

What it is:
Fraudulent emails or websites designed to steal login credentials or data.

Indicators:

Suspicious links or domains
Spelling/grammar errors
Urgent or threatening language

Defence:

Check sender email carefully
Hover over links before clicking
Use email filtering tools
🔹 Malware & Ransomware

What it is:
Malicious software designed to damage, disrupt, or gain unauthorised access.

Types:

Virus
Trojan
Spyware
Ransomware

Detection:

Slow system performance
Unknown processes running
Unexpected pop-ups

Mitigation:

Antivirus software
Regular updates
Avoid downloading unknown files
🔹 Passwords & Authentication

Best Practices:

Use strong, unique passwords
Avoid password reuse
Use password managers

Risks:

Brute force attacks
Credential stuffing
🔹 Multi-Factor Authentication (MFA)

What it is:
Adds an extra layer of security beyond passwords.

Examples:

SMS codes
Authenticator apps
Biometrics

Benefit:

Prevents unauthorised access even if passwords are compromised
🔹 Public Network Safety

Risks:

Man-in-the-middle attacks
Packet sniffing

Mitigation:

Avoid sensitive activity on public Wi-Fi
Use VPNs
🔹 Backups

Why important:

Protects against ransomware and data loss

Best Practice:

Follow 3-2-1 rule (3 copies, 2 media types, 1 offsite)
🔹 Updates & Patching

Why important:

Fixes vulnerabilities

Risk if ignored:

Systems become easy targets for attackers
🧪 Real-World Application

From this room, I learned how attackers exploit both technical vulnerabilities and human behaviour, and how basic security practices can significantly reduce risk.

🏁 Key Takeaway

Cybersecurity is not just about tools — it also involves user awareness, good practices, and layered defence.
